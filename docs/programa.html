Programa da Disciplina

Introdução à Ciência de Dados 2025
Bacharelado em Administração

Autor

Prof. Dr. Washington S. da Silva

Objetivo Geral

Esta disciplina cobre os fundamentos de ciência de dados aplicados à administração, desde estatística e probabilidade até programação e aplicações práticas, para equipar o graduando com um sólido conhecimento básico para trabalhar com análise de dados em contextos empresariais. O curso fornece uma introdução aos princípios do processo de análise de dados, com exemplos práticos sobre como aplicá-los em decisões. São apresentados conceitos essenciais de manipulação de dados, estatística descritiva, visualização e fundamentos de probabilidade e inferência, proporcionando ao estudante habilidades críticas para analisar dados de forma eficaz na era da Ciência de Dados. O objetivo final é preparar os graduandos para o mercado de trabalho atual, tornando-os capazes de coletar, preparar, analisar e comunicar insights baseados em dados em um contexto de negócios.

Objetivos Específicos

  1. Capacitar o graduando em ferramentas e técnicas modernas essenciais para Ciência de Dados, com foco na linguagem R e no ambiente RStudio para a importação, organização, visualização e análise estatística de dados.

  2. Desenvolver habilidades de preparação e manipulação de dados usando o metapacote tidyverse, capacitando o estudante a transformar dados brutos em formatos adequados para análise.

  3. Aplicar conceitos fundamentais de estatística descritiva e visualização de dados para identificar padrões e tendências relevantes em conjuntos de dados.

  4. Introduzir fundamentos de probabilidade e inferência estatística necessários para a tomada de decisão baseada em dados, incluindo simulações de Monte Carlo aplicadas a problemas de negócios.

  5. Capacitar o graduando para a criação de relatórios e dashboards usando o sistema Quarto, permitindo a reprodutibilidade das análises e a comunicação eficaz de resultados analíticos.

Ementa

Fundamentos de Ciência de Dados aplicados à Administração. Ambiente de trabalho e ferramentas computacionais (RStudio, Linguagem R, Git, GitHub e Sistema Quarto). Preparação, manipulação e transformação de dados com tidyverse. Estatística descritiva. Visualização de dados com ggplot2. Fundamentos de probabilidade. Introdução à inferência estatística. Criação de relatórios e dashboards para comunicação de resultados.

Avaliação

Os pontos referentes às avaliações serão distribuídos da seguinte forma:

  • Avaliação 1 (33 pontos): Fundamentos de R e Manipulação de Dados

    • Aplicada após o módulo 3
    • Projeto individual: importação e manipulação de dados empresariais
    • Entrega: Documento Quarto via GitHub

  • Avaliação 2 (33 pontos): Estatística Descritiva e Visualização

    • Aplicada após o módulo 5
    • Projeto em duplas: análise exploratória e visualizações
    • Entrega: Documento Quarto com análises e visualizações comentadas

  • Avaliação 3 (34 pontos): Projeto Final

    • Aplicada ao final do curso
    • Projeto em grupos de 3-4 alunos: análise completa de um problema aplicado
    • Entrega: Relatório Quarto ou Dashboard analíticos

Para aprovação, é necessária a obtenção de pelo menos 60 pontos. Pode haver alterações no conteúdo das avaliações conforme o andamento da disciplina e rendimento da turma.

Recursos Didáticos

Entre os recursos didáticos utilizados na disciplina estão: Slides e notas de aula, tutoriais desenvolvidos em Quarto, aulas computacionais práticas listas de exercícios aplicados, conjuntos de dados reais e simulados para análise, repositório GitHub da disciplina para compartilhamento de materiais e submissão de atividades, e ambiente virtual de aprendizagem para suporte.

Metodologia de Ensino

Trata-se de um curso do tipo “hands-on” e baseado no conceito de aprendizagem ativa. Nesse sentido, e dados os objetivos do curso, faremos uso intensivo de aulas práticas nas quais os graduandos deverão praticar as metodologias para análise de dados e sua implementação via linguagem R e sistema de publicação Quarto. O aprendizado será estruturado por meio de projetos progressivos, começando com exercícios simples de manipulação de dados e avançando para análises mais complexas, culminando em um projeto final que sintetiza as competências desenvolvidas ao longo do semestre. Esta abordagem visa facilitar a aplicação imediata dos conhecimentos no contexto profissional dos estudantes.
